# Twitchのチャットの外国語を翻訳するBOT

## 主な特徴

### 外国語と判定された時だけ翻訳します

外国語ではない場合は翻訳しないため、翻訳サービスの使用量が節約できます。

### 合成音声による読み上げ

※ 要 AssistantSeika

- VOICEVOX, COEIROINKなどの合成音声での読み上げを行います
- 翻訳された場合、翻訳結果のみ読み上げを行います
- 同じ人が連続で発言した場合、お名前の読み上げを省略します
- ユーザー毎に読み上げ音声を変える事もできます

※ 要 わんコメ

- わんコメで設定したニックネームを使ってお名前を読み上げます。

これらの特徴に恩恵を受けないなら[チャット翻訳ちゃん](https://www.sayonari.com/trans_asr/)という優れたアプリがあるのでそちらを使ってください。

## ダウンロード

こちらの[リリースページ](https://github.com/natukin1978/twitch-chat-trans-bot/releases)にて`TwitchChatTransBot-vX.XX.zip`というファイルをダウンロードしてください。

## 設定

`ConfigApp.exe`を実行して入力を進める事で基本設定する事ができます。
こちらの[設定に必要な認証情報の作り方](https://github.com/natukin1978/twitch-chat-trans-bot/wiki)にて具体的な手順が説明してあります。

`DetailConfigApp.exe`では、詳細な設定ができます。こちらの変更はアプリの再起動を必要とせず即座に反映されます。

## 実行

`TwitchChatTransBot.exe`を実行します。
黒い画面が表示されます。特にエラーメッセージのようなものが出ていなければ動いていると思います。

## トラブルシューティング

### WindowsによってPCが保護されました

得体のしれないアプリという事でOSによって保護されました。
このアプリを信用する場合は実行を許可してあげてください。
